# sessions/ — Session Logs

Store a brief log after every Claude session.

## File Naming

`YYYY-MM-DD-session-N.md`

## Template

```markdown
# Session [N] — YYYY-MM-DD

## Steps Completed
- Step [N]: [name] — [one-line result]

## Key Output
[one sentence — what was the most important thing produced]

## Problems Hit
[any blockers or bugs encountered]

## Next Session Goal
[one sentence — what to do first next time]

## Commit SHA
[short SHA from GitHub]
```
