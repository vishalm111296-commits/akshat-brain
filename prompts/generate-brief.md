# generate-brief.md — Use When Starting a Sub-Project

Paste this into Claude when starting a new sub-experiment or module:

---

```
I am starting a new sub-project within Akshat AI System.

Sub-project name: [NAME]
Description: [2 sentences about what this does]

Generate these files ready to paste into GitHub:
1. A PROJECT_BRIEF.md section for this sub-project
2. An updated CHECKPOINT.md with this as the next step
3. A SPEC.md entry for the new module

Format each as a clearly labeled code block.
Keep every file under 200 words.
```
